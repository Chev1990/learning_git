Формат:

- Нажать на +

Файлы загружены, отображаются на форме

- Нажать на +
Откроется окно с выбором загружаемого файла 
- выбрать 3 файла для загрузки, 1 файл одного из формата jpeg, png и прочее, 2 файла - формата doc, txt, pdf, xls и прочее
1 файл формата jpeg, png загружен, отображается на форме, 
2 других файла не загружены 
Появляется окно с предупреждением о том, что выбраны файлы запрещенного формата. Повторите загрузку, выбрав файл формата jpeg, png и т.д.

- На форме отображены 3 загруженных файла

- Нажать на +
Откроется окно с выбором загружаемого файла 
- выбрать 2 файла для загрузки одного из формата, отличного от формата картинок - jpeg, png и прочее
Файлы не загружены,не отображаются на форме. Появляется окно с предупреждением о том, что выбраны файлы запрещенного формата. Повторите загрузку, выбрав файл формата jpeg, png и т.д.

Ошибка может быть и на стороне бэка, то есть размер на стороне front может допускать файлы размера больше 20 mb, а на стороне бэка такая загрузка может все поломать.

- Нажать на +

Удаление файла:

3. Цена:

- Проверить ввод данных в поле Цена:
  - ввести значение 783
    Данное значение вводится
  - ввести значения по очереди: аппп, а456)б, 786уууу, %89г
    Данное значение не сохраняется в поле. Предупреждение о введенном формате.
  - проверить max число знаков до точки(предположим формат ввода нецелых чисел через точку вводятся) и после точки 2(предположим только 2 знака после зарятой допустимо) знака. Данное число сохранится в поле Цена. Проверить отображение на форме, все ли введенное число корректно отображается.
   - ввести значение с запятой. Данное значение не сохраняется в поле. Предупреждение о введенном формате.
   - ввсети значение с 3 знаками после точки. Данное значение не сохраняется в поле или значение обрезается до 2 знаков после запятой с предупреждением.

5. Флажок 

Расчет по курсу.

1 кейс:

Автоматический расчет.

1. Указать в поле Price любое числовое значение, например 2000 
2. В поле Currency выбрать любую валюту, например $
3. Выставить галочку в checkbox Auto Calculate 

Ожидаемое поведение: 
1. Рассчитана корректно каждая цена в соответствующей валюте по обменному курсу
2. Каждое поле заполнено
3. Поле с выбранной валютой, в качестве основной, подсвечено синим цветом, в нашем случае ячейка с $
4. Поля с ценами в данном случае запрещены на ввод

4. Ввести фиксированную цену для каждой валюты или оставить поле пустым, если товар не продается 

Ожидаемое поведение: 
1. Поле с выбранной валютой, в качестве основной, подсвечено синим цветом,в нашем случае ячейка с $ и автоматически заполнено указанным значением в поле Price
